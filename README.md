# E-Cookbook

A lightweight desktop recipe manager for Windows with a dark theme and customizable accent colors

## Features

- **Recipe Management**: Create, edit, and delete recipes
- **Rich Text Support**: Add ingredients, instructions, and descriptions
- **Image Support**: Attach images to your recipes
- **Customizable Theme**: Choose from 6 accent colors (Lime, Yellow, Orange, Red, Pink, Blue)
- **Search**: Alphabetically sorted recipe list
- **Storage**: All data stored locally in JSON format
- **Title Bar**: Borderless window design

## Download

1. Download "E-Cookbook.exe" from the Releases page
2. Double-click to run, no installation required
3. The app will create its data folder automatically in "%AppData%\E-Cookbook"
